## Project Overview

The Smart Student Housing Platform is a web-based application designed to help students find safe, affordable, and reliable accommodation.

Unlike traditional housing platforms, this system focuses on transparency, trust, and intelligent decision-making by integrating smart features such as lease analysis, hidden charge detection, and roommate matching.

---

## Problem Statement

Students often face several challenges while searching for PGs or rental rooms:

* Hidden charges not clearly mentioned
* Fake or misleading property listings
* Lack of trust in property owners
* No clarity on fair rent pricing
* Difficulty in finding compatible roommates

---

## Solution

This platform provides a smart and secure housing solution by combining:

* Intelligent analysis
* Verified information
* User-friendly interface

This helps students make informed and confident decisions.

---

## Key Features

### Property Listings

* Browse available rooms and PGs
* Apply filters based on price, location, and type

### Smart Map (Nearby Places)

* Displays nearby colleges, shops, and transport
* Helps users evaluate location convenience

### Lease Analyzer (PDF Upload)

* Upload rental agreements
* Extracts and analyzes important clauses

### Hidden Charges Detection

* Detects electricity charges, maintenance fees, deposits, and penalties
* Alerts users about extra costs

### Roommate Matching

* Matches users based on budget, cleanliness, and lifestyle habits
* Provides compatibility score

### Rent Estimator

* Estimates fair rent based on location and room type
* Helps avoid overpaying

### Report Listing

* Users can flag suspicious or fake listings
* Improves platform safety

### Contact Owner

* Direct communication via call or WhatsApp

### Smart Recommendations

* Suggests similar properties based on user preferences

### Anonymous Reviews

* Users can rate and review properties
* Builds trust and transparency

## Installation and Setup

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/student-housing-platform.git
cd student-housing-platform
```

### 2. Install Dependencies

```bash
pip install flask flask-login flask-sqlalchemy pdfplumber requests
```

### 3. Configure Database

* Create a MySQL database
* Update credentials in `app.py`

### 4. Run the Application

```bash
python app.py
```

### 5. Open in Browser

```
http://127.0.0.1:5000
```
